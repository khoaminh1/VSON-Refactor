# Icons

Files this folder should contain (expected by `base-head.html`):

| File | Purpose | Size |
|---|---|---|
| `favicon.ico` | Browser tab icon | 32×32 |
| `icon.svg` | Scalable app icon (used by modern browsers / PWA) | any |
| `apple-touch-icon.png` | iOS home-screen icon | 180×180 |

## Source

The Forest Green "V" lockup from the nav logo is the canonical source. Export
from Figma (or equivalent) at the three sizes above.

## Current state

Stubs only — production assets need to be generated from the brand package
before launch. Until then the links 404 silently.
