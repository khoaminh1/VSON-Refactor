## What to Request From Each Brand

When reaching out to each partner, request the following:

**1. Logo file**
- Preferred format: SVG (vector) — if not available, PNG with transparent background
- Minimum size: 400px wide
- Preferred: both a colour version and a reversed/white version if available
- File naming: `torayvino-logo.svg`, `dell-logo-white.png`, etc.

**2. Written approval**
A short email confirmation is sufficient. It should state:
- The brand name
- Permission for VSON to display the logo on vson.vn
- Any usage restrictions (e.g. colour rules, minimum size, clear space)

A template message is provided below.
